# Heart Disease Prediction App

## About the Project
This project is a machine learning-based web application that predicts the likelihood of heart disease using medical input parameters. It demonstrates how ML models can be integrated into a simple web interface for real-world use.

The application allows users to enter health-related details and instantly receive prediction results through an interactive dashboard.

## Highlights
- Machine learning-powered prediction system
- Interactive web interface using Streamlit
- Instant results based on user input
- Lightweight and easy to use
- Beginner-friendly project for ML deployment

## Tech Stack
- Programming Language: Python
- Libraries: Pandas, NumPy, Scikit-learn
- Framework: Streamlit
- Version Control: Git & GitHub
